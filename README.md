# Netflix-Content-Analysis
Data-driven analysis of Netflix movies and TV shows dataset to identify genre trends, rating distribution, country-wise production, and content growth insights.


# Project Overview
Performed Exploratory Data Analysis (EDA) on Netflix dataset to understand content trends, ratings distribution, country-wise production, and release patterns.

# Objective
To analyze Netflix content and extract business insights that help understand user preferences and content strategy.

# Tools & Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

# Data Cleaning Process
Handled missing values
Removed unnecessary columns
Converted date columns into datetime format
Split multi-value columns like genre and country

# Key Analysis Performed
Content Type Distribution (Movies vs TV Shows)
Country-wise Content Production
Ratings Distribution
Year-wise Content Growth
Top Genres Analysis

# Key Insights
Movies dominate Netflix compared to TV Shows
USA produces the highest number of titles
Most content added after 2015
TV-MA rating is most common

# Business Conclusion
Netflix focuses heavily on movie content and mature audience category. Rapid content growth after 2015 indicates aggressive expansion strategy.
